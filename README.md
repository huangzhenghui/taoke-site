# taoke-site

Next.js + TypeScript project skeleton for the taoke content site.

## Development

Install dependencies:

```bash
pnpm install
```

Start the local development server:

```bash
pnpm dev
```

Open http://localhost:3000 in your browser.

## Scripts

```bash
pnpm lint
pnpm build
pnpm start
```

## Current Scope

This repository currently contains only the base Next.js project skeleton and planning documents.

No business features, database, Prisma, Redis, or external API integrations have been added yet.
